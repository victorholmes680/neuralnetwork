# neuralnetwork
Simple stb-style head-only library for Neural Networks. Created for educational purpose. Not suitable for production.

## Quick Start

```console
$ ./build.sh
$ ./adder_gen
$ ./gym ./adder.arch ./adder.mat
```

## Gym

Gym is a separate GUI app that uses neuralnetwork to pre-train Neural Networks.

## Controls
use space the pause the process and use r to restart the training
