\section{Gradient Descent}
The partial derivative of the cost function with respect to parameter $w_i$ can be approximated using finite differences:
\begin{align}
  \frac{\partial C}{\partial w_i} &= \frac{C(w_i + \epsilon) - C(w_i)} {\epsilon}
\end{align}
